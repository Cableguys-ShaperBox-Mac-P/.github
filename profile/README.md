## 💡 Overview

<a href="#cableguysShaperBox">Cableguys ShaperBox</a> is a multi-effect modulation plugin for Mac DAWs that creates rhythmic audio effects through drawable, tempo-synchronized LFO shapes applied to volume, filter frequency, stereo panning, and time-based parameters — the creative rhythm shaping and tremolo effects that define contemporary electronic music production.

The <a href="#cableguysShaperBox">drawable LFO shape system</a> is ShaperBox's core innovation — rather than selecting from a library of preset waveform shapes (sine, square, sawtooth), the user draws the modulation shape directly in a waveform editor with freehand drawing, snap-to-grid point placement, and curve handle manipulation. Any modulation shape imaginable draws directly: smooth curves, sharp steps, complex multi-peak patterns, asymmetric rhythms — the modulation behavior exactly matches the drawn shape applied at the configured tempo-sync rate. The <a href="#cableguysShaperBox">tempo-synchronized modulation</a> locks the LFO cycle to the DAW's tempo — the modulation completes one cycle per bar, per half note, per quarter note, or any musical division, creating effects that stay rhythmically locked to the track without manual tempo calculation or drift over time.

The <a href="#cableguysShaperBox">multiple effect types</a> in ShaperBox 3 include VolumeShaper for amplitude modulation (tremolo, pumping, gating effects), FilterShaper for cutoff frequency modulation (rhythmic filter sweeps, wah effects), PanShaper for stereo positioning modulation (auto-pan, stereo movement), TimeShaper for sample timing modulation (stutter, glitch, tape flutter effects), and additional shapers for different parameter types. Each shaper type applies the drawable LFO concept to a different audio parameter, and multiple shapers chain together in a single ShaperBox instance. The <a href="#cableguysShaperBox">multi-band processing</a> splits the audio into frequency bands for independent modulation — applying volume shaping only to the low frequencies while leaving the high frequencies unaffected, or creating crossover-based modulation effects that affect different frequency regions differently.

<a href="#cableguysShaperBox">Side-chain input capability</a> routes an external audio signal as the modulation source instead of the drawn LFO — using a kick drum or rhythmic source to drive the modulation creates effects that respond dynamically to the sidechain input rather than a static drawn shape. <a href="#cableguysShaperBox">Parallel processing chains</a> run multiple ShaperBox effect instances in parallel within a single plugin — blending different modulation treatments of the same signal for complex layered effects.

<a href="#cableguysShaperBox">Preset library</a> includes professionally designed effect presets covering common production scenarios — pumping sidechains, rhythmic filter effects, auto-pan patterns, stutter and glitch effects — as starting points for common use cases that further customize with the drawable LFO. <a href="#cableguysShaperBox">AU and VST plugin formats</a> run ShaperBox in Logic Pro, Ableton Live, GarageBand, and other compatible Mac DAWs as an insert effect on any audio or instrument track.

ShaperBox's drawable LFO concept is the design decision that distinguishes it from conventional modulation plugins. Standard LFO plugins offer a selection of waveform shapes — sine, triangle, sawtooth, square, random — and the producer selects whichever approximates the desired effect. ShaperBox replaces that selection with direct drawing: the producer draws exactly the modulation shape that produces the intended effect, with no approximation required. The difference between a shape that almost does what you want and a shape that exactly does what you want is the difference between ShaperBox and conventional LFO tools.
